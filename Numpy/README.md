# 🔢 NumPy

This folder contains my practice and implementation of NumPy concepts used for numerical computing in Python.

## 📚 Topics Covered

- Array creation (`np.array`, `arange`, `linspace`)
- Indexing and slicing
- Array operations (addition, multiplication)
- Reshaping arrays
- Statistical functions (mean, sum, max, min)

## 🎯 Goal

To understand how numerical data is handled efficiently using NumPy arrays.

## 🧠 Key Learning

- NumPy arrays are faster than Python lists
- Vectorized operations improve performance
