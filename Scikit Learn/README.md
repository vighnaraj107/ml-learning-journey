# 🤖 Scikit-learn

This folder contains implementation of basic machine learning algorithms using Scikit-learn.

## 📚 Topics Covered

- Data preprocessing
- Train-test split
- K-Nearest Neighbors (KNN)
- Logistic Regression
- Model evaluation

## 🎯 Goal

To understand how machine learning models are trained and evaluated.

## 🧠 Key Learning

- ML workflow: data → train → predict → evaluate
- Choosing the right model is important
